## About

ToiCF is a research-oriented organization focused on edge networking, protocol experiments, and practical network tooling.

We explore how modern network protocols behave in real runtimes, how edge platforms handle transport workloads, and how to turn experimental findings into small reproducible tools.

## Focus

- Edge network behavior and runtime limits
- Proxy, tunnel, and relay protocol research
- Transport-layer measurement and optimization
- Real-link validation and reproducible diagnostics
- Practical tools built from verified experiments

## Approach

- Measure real behavior instead of guessing from labels.
- Build small testable artifacts before drawing conclusions.
- Keep experimental code honest about its limits.
- Prefer practical verification over purely descriptive checks.

## Notes

Most repositories are experiments or research tools. They may be useful for learning, validation, and engineering exploration, but are not always polished end-user products.
